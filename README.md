#  E-Commerce Delivery Analytics

## 📌 Overview
Interactive analysis of 100K orders across 
Blinkit, JioMart and Swiggy Instamart using 
Python and Power BI.

## Objective
To analyze delivery performance, revenue trends,
refund patterns and customer ratings across 
platforms and product categories.

## Dataset
- Source: Kaggle
- Rows: 100,000
- Columns: 11

## 🔍 Key Insights
- Total Revenue: ₹59M
- Total Orders: 100K
- Average Rating: 3.24/5
- Average Delivery Time: 29.54 min
- Average Order Value: ₹590.99
- Total Refunds: 46K
- Personal Care = Highest Revenue (₹17M)
- 54% orders had no refund requests
- Revenue evenly distributed across platforms

## 🛠 Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Power BI
- Jupyter Notebook
